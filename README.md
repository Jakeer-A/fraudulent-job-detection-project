# fraudulent-job-detection-project
Job Scam Detection using Machine Learning and Deep Learning with Python, Scikit-learn, TensorFlow/Keras, and the EMSCAD dataset.
# Comparative Analysis of Deceptive Job Advertisement Prediction Using Data Mining

An advanced data science repository evaluating traditional machine learning frameworks and deep learning paradigms to proactively identify and mitigate fraudulent online recruitment scam vectors.

---

## 📝 Project Abstract
In modern environments heavily reliant on digital social communication, malicious employment scams pose a severe threat to job seekers. This project implements and analyzes multiple classification topologies—including KNN, Decision Trees, Support Vector Machines (SVM), Naïve Bayes, Random Forest, Multilayer Perceptron (MLP), and Deep Neural Networks (DNN)—to predict if a vacancy listing is authentic or fraudulent.

Empirical evaluations conducted on the **Employment Scam Aegean Dataset (EMSCAD)** containing 18,000 samples demonstrate that while conventional ensemble strategies show high precision, a **3-layer Deep Neural Network architecture achieves an optimal ~98% classification accuracy**[cite: 2].

## 🔍 System Evolution Analysis

### ❌ The Existing Paradigm
Prior implementations frequently leverage standard classifiers like Zero R, One R, or basic Random Forest configurations on balanced data slices[cite: 2]. 
* **Disadvantages:** Traditional baselines are highly susceptible to deceptive data patterns designed to bypass simple text filtering, leaving critical professional and personal information vulnerable to exploitation[cite: 2].
* **Baseline Accuracy:** Typical benchmarks peak at approximately 89.5% accuracy under traditional workflows[cite: 2].

### 🚀 The Proposed Architecture
Our optimized approach structures data processing into a rigorous three-tiered sequence[cite: 2]:
1. **Data Pre-Processing:** Aggressive noise reduction and HTML tag stripping to preserve fundamental textual profiles[cite: 2].
2. **Feature Engineering:** High-efficiency categorical dimension reduction utilizing linear scaling[cite: 2].
3. **Multi-Algorithmic Classification:** Concurrent training across classical data mining models and a dense 3-layer feed-forward Deep Neural Network[cite: 2].

#### Key Advantages:
- Significantly minimizes the total parameter overhead and training runtime[cite: 2].
- Delivers a highly generalized model capable of processing real-time unseen inputs with up to 99% accuracy in targeted validation folds[cite: 2].
